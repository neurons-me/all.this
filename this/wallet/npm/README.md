# THIS.WALLET
Manage multiple **blockchains**, making it easy to organize and manage your crypto assets. The `Wallet` class is capable of creating accounts, initializing new blockchain wallets, and importing existing ones, all under the same account. This approach provides both flexibility and security, with the ability to manage multiple blockchains under one account name.

# Getting Started:
##### **Install `this.wallet`:**
Open your terminal and run the following command to install the `this.wallet` package:

```bash
npm install this.wallet
```

##### **Import `this.wallet` in Your Project:**
In the JavaScript file where you want to use `this.wallet`, import the `thisWallet` class.

```js
import wallet from 'this.wallet';
```
