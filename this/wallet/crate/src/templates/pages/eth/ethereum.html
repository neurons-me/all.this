Ethereum API Overview

The this.Wallet API allows you to interact with Ethereum-compatible networks through a secure JSON-RPC proxy. You can send requests to the configured RPC nodes and get back raw blockchain data.

Allowed Methods

For security and performance, the API only exposes a limited set of JSON-RPC methods during the MVP phase:

Method Description Try
eth_chainId Returns the current Ethereum chain ID (e.g. 0x1 for Mainnet). Test
eth_getBalance Fetches the balance of a given address in wei. Test
eth_call Executes a read-only call to a smart contract without sending a transaction. Test
eth_estimateGas Estimates the gas needed to execute a transaction. Test
eth_getTransactionCount Returns the nonce for an address (number of sent transactions). Test
eth_sendRawTransaction Broadcasts a signed transaction to the Ethereum network. Test
net_version Returns the current network ID in decimal (e.g. 1 for Mainnet). Test
web3_clientVersion Returns the name and version of the Ethereum client used by the node. Test
